CSS3:

hoja de estilos en cascada (Cascading Style sheets), este lenguaje se dirige al diseño gráfico, o visual de los sitios web, e interfaces de usuario HTML o XHTML, se puede trabajar en conjunto con JavaScript y en la misma hoja en donde se está desarrollando un documento HTML, pero para mayor productividad es preferible trabajarla por aparte en un documento extension.css. El cual se puede vincular a varias páginas HTML.

Css trabaja mediante módulos, los cuales son categorías en las que se dividen las modificaciones que se hacen al sitio web, algunos ejemplos son:”colors”, “fonts”, “backgrounds”.

Con esta herramienta podemos agregar animaciones y transiciones a los objetos de nuestra página, también jugar con los colores usando gradientes, o cambiar las posiciones y tamaños de los objetos, entre otras muchas ventajas graficas que nos ofrece este amplio y práctico lenguaje.
